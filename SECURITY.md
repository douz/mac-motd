# Security Policy

## Supported Versions

Security fixes are generally applied to the latest release.

## Reporting a Vulnerability

Please do not open a public issue for security vulnerabilities.

Report privately to:

- `dbarahona@me.com`

Please include:

- Affected version/tag
- Reproduction steps
- Impact assessment
- Suggested mitigation (if available)

We will acknowledge receipt and follow up with remediation details.
