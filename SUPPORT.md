# Support

## Scope

Community support is provided on a best-effort basis.

## Where to Ask for Help

- Open a GitHub issue for bugs and feature requests.
- Use issue templates to provide complete context.

## Response Expectations

- Initial triage target: within 7 days.
- Priority is given to regressions and installation failures.

## Security Issues

Do not file public issues for vulnerabilities.

Report privately via `SECURITY.md`.
